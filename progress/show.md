# /progress show

## 指令

读取当前项目的 `PROGRESS.md`，提取关键信息并格式化输出。

## 执行流程

1. 定位并读取 `PROGRESS.md`。
2. 解析文件，提取以下部分：
   - 🎯 当前聚焦的任务
   - 📥 待办队列的前 5 项
   - ⚡ 快速恢复步骤
3. 忽略已完成的详细列表，仅统计完成数量（如：✅ 最近完成: 3项）。
4. 若文件不存在，提示运行 `/progress checkpoint` 初始化。

## 输出格式

```
📊 **项目进度概览**
- **当前焦点**: [列出 🎯 下的任务]
- **下一步**: [列出 📥 下的前几项]
- **恢复指引**: [列出 ⚡ 下的步骤]
```
